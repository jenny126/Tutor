# 自訂函式(function)
```c=
int 自訂函式名稱(變數,變數,變數)
{
  內容
  return 要輸出的變數;
}
```
範例
```c=
#include <stdio.h>

int maxxx(int a, int b) {
    if (a > b) {
        return a;
    } else {
        return b;
    }
}

int main() {
    int a, b, c;
    int ans = 0;
    scanf("%d %d %d", &a, &b, &c);

    int temp = maxxx(a, b);
    int maxNumber = maxxx(temp, c);
    printf("%d",maxNumber);
    return 0;
}

```
